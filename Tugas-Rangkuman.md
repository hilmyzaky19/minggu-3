## Bab 2: Variabel & Tipe Data

- Variabel itu ibarat kotak buat nyimpen data.
- const dipakai kalau datanya tetap dan gak bakal berubah.
- let dipakai kalau datanya bisa berubah-ubah.
- Tipe data yang sering dipakai: String (teks), Number (angka), Boolean (true/false), Null, dan Undefined.
 ## Bab 3: Operator
- Operator itu alat buat ngolah data.
- Bisa buat ngitung angka, ngebandingin nilai, atau ngecek beberapa kondisi sekaligus.
- Hasil perbandingan biasanya berupa true atau false.

## Bab 4: Input & Output
- Input = data yang masuk ke program.
- Output = hasil yang keluar dari program.
- Program bisa nerima data dari user lalu nampilin hasilnya kembali.
- Kadang data input perlu diubah dulu sebelum diproses.

## Bab 5: Sintaks Dasar
- JavaScript bedain huruf besar dan kecil.
- Komentar dipakai buat kasih catatan di kode.
- Biasain nulis kode yang rapi biar gampang dibaca.
- Utamain pakai const, kalau perlu diubah baru pakai let.

## Bab 6: Percabangan
- Percabangan dipakai buat ngambil keputusan.
- Kalau kondisi tertentu terpenuhi, program jalan ke jalur A, kalau enggak ke jalur B.
- if, else if, dan else dipakai buat bikin logika keputusan.
- switch cocok buat banyak pilihan yang nilainya udah jelas.
- Truthy = dianggap benar, Falsy = dianggap salah.

## Bab 7: Looping
- Looping dipakai buat ngulang pekerjaan otomatis.
- for kalau jumlah pengulangannya udah jelas.
- while kalau belum tahu bakal ngulang berapa kali.
- do...while minimal jalan sekali.
- break buat berhenti total.
- continue buat skip satu putaran.
- Hati-hati infinite loop, bisa bikin program muter terus tanpa berhenti.

## Bab 8: Array
- Array itu tempat nyimpen banyak data dalam satu variabel.
- Data di dalam array punya nomor urut yang disebut indeks dan dimulai dari 0.
- Bisa nambah, hapus, cari, atau ambil data dengan berbagai method bawaan JavaScript.
- Salah satu struktur data yang paling sering dipakai.

## Bab 9: Function
- Function itu kumpulan kode yang bisa dipakai berulang kali.
- Bikin sekali, pakai berkali-kali.
- Bisa nerima data dari luar lewat parameter.
- Bisa ngasih hasil balik lewat return.
- Bikin kode jadi lebih rapi dan gak banyak copas.

## Bab 10: Object
- Object dipakai buat nyimpen data yang saling berhubungan.
- Isinya berupa pasangan key dan value.
- Bisa nyimpen data sekaligus function (method).
- this dipakai buat ngakses data milik object itu sendiri.
- Cocok buat nyimpen data profil, produk, akun, dan sejenisnya.

## Bab 11: Iterable & Non-Iterable
- Iterable = data yang bisa disisir satu-satu.
- Contohnya Array dan String.
- Biasanya diproses pakai for...of.
- Object bukan iterable.
- Kalau mau nyisir Object pakai for...in.
- for...of ambil nilai, for...in ambil nama properti (key).
